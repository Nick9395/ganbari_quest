# アプリ名 がんばりクエスト

## サービス概要（どんなサービスなのかを３行で説明してください。）

このアプリは児童向けの学習ログアプリです。目標や得点ルールを自分で設定し、「がんばったボタン」を押して経験値を集めるとレベルが上がります。日々のがんばりをロールプレイングゲーム感覚で楽しく記録し、成果を可視化して振り返りをすることもできます。

## このサービスへの思い・作りたい理由（このサービスの題材となるものに関してのエピソードがあれば詳しく教えてください。このサービスを思いつくにあたって元となる思いがあれば詳しく教えてください。）

- 自宅学習などの自己管理が苦手な長女がおり、いつも怒られています。自分も幼少時代を思い返すと、宿題もせずに友人とゲームばかりしていました。学習に取り組むことが苦手な子どもと、現代の忙しい親世代、双方の手助けになればと思い、このアプリを作ろうと思いました。
- 子どもが使っていた「できたねシール」や「ラジオ体操カード」、自分が子どもの頃好きだった「ロールプレイングゲーム」から着想を得ました。

## ユーザー層について（決めたユーザー層についてどうしてその層を対象にしたのかそれぞれ理由を教えてください。）

- 自分や自分の子どもに役立つアプリを作ろうと思ったから。
- 子どもの学力を気に掛ける度合いが高くなっているので、学習への取り組みを助けるアプリは需要があると思ったから。
- IT教育が進んでいるので、児童が使用しやすいと考えたから。

## サービスの利用イメージ（ユーザーがこのサービスをどのように利用できて、それによってどんな価値を得られるかを簡単に説明してください。）

- その日一日の成果（宿題や自主学習、家の手伝いなど）と自分で決めた評価基準に準じてボタンをクリックする。使用する端末は自分のPCやタブレット、親のスマートフォンなど。
- 子どもが自分から家庭学習に取り組むきっかけになる。
- 目標設定、計画、実施、振り返りといった考え方の練習になる。
- 自発的な学習への取り組みを継続できるようになると、親の負担軽減にもなる。

## ユーザーの獲得について（想定したユーザー層に対してそれぞれどのようにサービスを届けるのか現状考えていることがあれば教えてください。）

- 足を使って教育機関等に営業をかける。
- 知人への口コミ。
- その他web上での広報活動。

## サービスの差別化ポイント・推しポイント（似たようなサービスが存在する場合、そのサービスとの明確な差別化ポイントとその差別化ポイントのどこが優れているのか教えてください。独自性の強いサービスの場合、このサービスの推しとなるポイントを教えてください。）

- 日々のがんばりをロールプレイングゲーム感覚で楽しく記録し、振り返りもできる。
- 学習に限らず、取り組むべき内容を自由に設定できる。（例：家の手伝い、習い事、課外活動など）
- 児童に限らず、日々のがんばりを手間暇かけずに楽しく記録したいすべての人が利用できる。
- PDCAサイクルの考え方を練習することができる。

## 機能候補（現状作ろうと思っている機能、案段階の機能をしっかりと固まっていなくても構わないのでMVPリリース時に作っていたいもの、本リリースまでに作っていたいものをそれぞれ分けて教えてください。）

【確実に実装したい機能】

- カレンダー機能（入力したい日付をクリックで入力画面に遷移）
- ボタンを押してがんばりを記録する機能（経験値アップ、レベルアップ、ステータスアップ、効果音など）
- ボタンを押して怠けた分を減点する機能（経験値ダウン、レベルダウン、ステータスダウン、効果音など）
- 月の目標設定、基準設定、振り返りなどを入力する画面に遷移
- がんばりの可視化（得点のグラフ化、ユーザーのレベルやステータス表示など）

【できれば実装したい機能】

- ユーザー機能（ユーザー登録、ログイン、パスワードリセットなど）
- アドマイン機能
- 管理者から児童へコメントできる機能
- 月ごとのがんばりを表彰状（PDF）に出力して印刷できる機能
