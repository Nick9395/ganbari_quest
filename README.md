# アプリ名:がんばりクエスト(ganbari_quest)

## サービス概要（どんなサービスなのかを３行で説明してください。）

このアプリは学童期の児童を対象とした自主学習ログアプリです。目標や得点ルールを自分で設定し、がんばった分だけ「がんばった」コマンドを押してモンスターを倒します。
モンスターを倒すと経験値が手に入り、集めた経験値に応じてユーザーのレベルが上がります。逆に「なまけた」コマンドを押すと経験値もレベルも下がってしまいます。がんばって経験値を集めましょう。
日々のがんばりはグラフ化され、成果を振り返ることもできます。ロールプレイングゲームのような雰囲気で、楽しみながら学習習慣を身につけましょう。

- 自宅学習や自己管理が苦手な長女がおり、いつも怒られています。自分も幼少時代は宿題もせずに友人とゲームばかりしていました。自宅学習が苦手な子どもと忙しい親世代、双方の手助けになればと思い、このアプリを作ろうと思いました。
- 子どもが使っていた「できたねシール」や「ラジオ体操カード」、自分が子どもの頃好きだった「ロールプレイングゲーム」などから着想を得ました。

## サービスの利用イメージ（ユーザーがこのサービスをどのように利用できて、それによってどんな価値を得られるかを簡単に説明してください。）

- ユーザー情報を入力してログインするとメイン画面へ遷移する。
- メイン画面にはカレンダーやステータス、目標や評価基準、各種設定画面へのリンクなどが表示される。
- 月初め（または前月）に各種設定画面にて該当月の目標を入力する。（例：授業でやったところの練習問題をすべてやる、漢字ドリルを終わらせるなど）
- 月初め（または前月）に各種設定画面にて該当月の評価基準（例：ドリル１ページやったら「がんばった」コマンド２回、宿題忘れたら「なまけた」コマンド１回など）を決めて入力する。（次月以降は最初に決めた評価基準が自動転記される。変更したければ編集する）
- カレンダーの該当日をクリックするとバトル画面へ遷移。評価基準で決めたぶんだけ「がんばった」コマンドを押して、モンスターを倒す。
- 「がんばった」コマンドを押すたびに「モンスターを倒した」といったメッセージが表示されて経験値が得られる。(最大7984Exまで)逆に「なまけた」コマンドを押すたびに経験値は下がる。(最小1Lvまで)
- 経験値が8の倍数まで上がるたびにレベルが上がる。「レベルが上がった」というメッセージが表示されてレベルアップの効果音が鳴る。逆に経験値が一定数を下回るとレベルが下がり、レベルダウンの効果音が鳴る。(最大999レベル、最小1レベル)
- このサービスを利用することで、子どもが自主的に学習に取り組むきっかけになる。
- 自分ががんばったことを視覚化して振り返りを行うことで、学習への動機づけをより内発的なものへと変化させることができる。その結果、学習習慣を定着させられる可能性がある。
- 目標設定、実施、振り返りまでのサイクルを繰り返す仕組みになっているので、課題解決のための思考を身につけられる。
- 子どもが自主的に学習に取り組む習慣を身につけることができるので、親の負担軽減にもなる。

## 機能候補（現状作ろうと思っている機能、案段階の機能をしっかりと固まっていなくても構わないのでMVPリリース時に作っていたいもの、本リリースまでに作っていたいものをそれぞれ分けて教えてください。）

【MVPリリースまでに実装したい機能】

- ユーザー機能（ユーザー登録、ログイン、ログアウト）
- カレンダー機能（入力したい日付をクリックで入力画面に遷移）
- 「がんばった」コマンドを押すたびに経験値を獲得できる機能（経験値の加算、経験値アップのフラッシュメッセージ）
- 経験値が8の倍数までたまるごとにレベルが1上がる機能(経験値とレベルの加算、レベルアップのフラッシュメッセージ、効果音)
- 「なまけた」コマンドを押すたびに経験値が減算される機能（経験値の減算、経験値ダウンのフラッシュメッセージ）
- 経験値が8の倍数を下回るごとにレベルが1下がる機能(経験値とレベルの減算、レベルダウンのフラッシュメッセージ、効果音)
- バトル画面のフラッシュメッセージが、表示後５秒くらいで消える機能
- 月の目標設定、評価基準設定、振り返り入力機能
- 該当月に獲得した経験値をグラフ化する機能

【MVPリリース後に追加予定の機能、できれば実装したい機能】

- 獲得した経験値を月別にグラフ化する機能
- レベルに応じてステータスを表示する機能（攻撃力、防御力、魔力など）
- ユーザー機能（パスワードリセットなど）
- アドマイン機能(所属グループ内のユーザーのデータを編集、削除、コメント)
- 月ごとのがんばりを表彰状（PDF）に出力して印刷できる機能
- PWA化
- アバター機能（ユーザーのホーム画面にアバターを表示、レベルに応じて装備がパワーアップ）
- アプリの使い方ページの目次遷移機能
- バトル画面にモンスター的なものを表示する機能。
- バトル画面のモンスター的なものが、バトル終了のフラッシュメッセージが消えるタイミングで別のモンスターの画像にランダムで変わる機能。
