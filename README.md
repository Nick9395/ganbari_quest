# アプリ名:がんばりクエスト(ganbari_quest)

## サービス概要（どんなサービスなのかを３行で説明してください。）

このアプリは学童期の児童を対象とした自主学習ログアプリです。月ごとに目標や得点ルールを自分で設定し、「がんばったボタン」を押して経験値を集めます。
集めた経験値に応じてレベルが上がります。「なまけたボタン」を押すと経験値もレベルも下がってしまいます。がんばって経験値を集めましょう。
日々のがんばりはグラフ化され、成果の振り返りもできます。自主学習の習慣や自己管理の流れをロールプレイングゲーム感覚で楽しく身につけましょう。

## このサービスへの思い・作りたい理由（このサービスの題材となるものに関してのエピソードがあれば詳しく教えてください。このサービスを思いつくにあたって元となる思いがあれば詳しく教えてください。）

- 自宅学習や自己管理が苦手な長女がおり、いつも怒られています。自分も幼少時代は宿題もせずに友人とゲームばかりしていました。自宅学習が苦手な子どもと忙しい親世代、双方の手助けになればと思い、このアプリを作ろうと思いました。
- 子どもが使っていた「できたねシール」や「ラジオ体操カード」、自分が子どもの頃好きだった「ロールプレイングゲーム」などから着想を得ました。

## サービスの利用イメージ（ユーザーがこのサービスをどのように利用できて、それによってどんな価値を得られるかを簡単に説明してください。）

- ユーザー情報を入力してログインするとメイン画面へ遷移する。
- メイン画面にはカレンダーやステータス、目標や評価基準、各種設定画面へのリンクなどが表示される。
- 月初め（または前月）に各種設定画面にて該当月の目標を入力する。（例：授業でやったところの練習問題をすべてやる、漢字ドリルを終わらせる）
- 月初め（または前月）に各種設定画面にて該当月の評価基準（例：ドリル１ページやったら「がんばったボタン」２回、宿題忘れたら「なまけたボタン」１回）を決めて入力する。（次月以降は最初に決めた評価基準が自動転記される。変更したければ編集する）
- 自分へのプチごほうびを入力する。（例：２回がんばった日はyoutube１時間みられる、毎日がんばったら土曜日のおやつはアイス）
- カレンダーの該当日をクリックすると戦闘画面へ遷移。自分で決めた評価基準に合わせて「がんばったボタン」や「なまけたボタン」を押す。
- 「がんばったボタン」を押すたびに「モンスターを倒した」といったメッセージが表示されて経験値が得られる。「なまけたボタン」を押すたびに経験値は下がる。
- 経験値が一定数まで上がるたびに「レベルアップした」といったメッセージが表示されてレベルアップの効果音が鳴る。経験値が一定数を下回るとレベルが下がり、レベルダウンの効果音が鳴る。
- このサービスを利用することで、子どもが自主的に学習に取り組むきっかけになる。
- 自分ががんばったことを視覚化して成果の振り返りを行うことで、学習への動機づけをより内発的なものへと変化させることができる。その結果、学習習慣を定着させられる可能性がある。
- 目標設定して、学習を実施して、それを振り返るまで流れを身につける練習になる。
- 子どもが自主的に学習に取り組む習慣を身につけられると、親の負担軽減にもなる。

## サービスの差別化ポイント・推しポイント（似たようなサービスが存在する場合、そのサービスとの明確な差別化ポイントとその差別化ポイントのどこが優れているのか教えてください。独自性の強いサービスの場合、このサービスの推しとなるポイントを教えてください。）

- 日々のがんばりをロールプレイングゲーム感覚で楽しく数値化して記録することができる。
- 目標設定や評価基準を自分で設定して振り返りもする仕組みなので、賞罰などの外発的な動機づけから自己実現的な内発的動機づけへと意識を変えていくことができる。

- 児童に限らず、日々のがんばりを手間暇かけずに記録したいすべての人が利用できる。（例：ジョギング、ジム通い、資格の勉強など）
